About

             Made using Python and Qt, CtrlCat has the goal of helping users to manage text copies made using Ctrl+C. This application clipboard's copies are automatically saved inside a SQL data bank, allowing users to make changes to them and select from previous copies through a key sequence.

Download

             CtrlCat is free and currently available for Windows and Debian's Linux.
